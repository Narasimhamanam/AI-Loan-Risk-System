# 💎 AI Loan Risk Analysis Dashboard

A premium AI-powered Loan Risk Prediction Dashboard built using **Streamlit**, **Machine Learning**, and **Modern Fintech UI Design**.

This project predicts whether a loan application is likely to be **Approved**, **Rejected**, or **Under Review** using applicant financial data and intelligent risk analysis.

# 📌 Features

## ✅ AI Loan Approval Prediction
- Predicts loan approval probability
- Classifies applications into:
  - Approved
  - Rejected
  - Under Review

---

## ✅ Intelligent Risk Analysis
The system analyzes:
- Applicant Income
- Credit History
- Loan Amount
- Education
- Marital Status

Then calculates:
- Approval Probability
- Risk Level
- AI Decision

---

## ✅ AI Explanation System
Provides:
- Why application was approved/rejected
- Financial risk explanation
- AI-generated recommendations
- Suggestions to improve approval chances

---

# 🧠 AI Decision Logic

The prediction system evaluates:

| Feature | Impact |
|---|---|
| Credit History | Very High |
| Applicant Income | High |
| Loan Amount | Medium |
| Education | Medium |
| Marital Status | Low |

---

# 📊 Risk Classification

| Probability | Risk Level |
|---|---|
| 75% - 100% | Low Risk |
| 50% - 74% | Medium Risk |
| Below 50% | High Risk |

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Loan-Risk-System.git
```

---

## 2️⃣ Navigate to Project

```bash
cd AI-Loan-Risk-System/frontend
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Streamlit App

